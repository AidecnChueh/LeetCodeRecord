## **6. 二分搜尋（Binary Search）**
**📌 內容**：適用於排序數據的高效搜尋  
**🔹 常見題型**：
- **標準二分**：搜尋目標數
- **變形二分**：搜尋旋轉數組、最接近值
- **左/右界搜尋**：找最左/最右出現位置
