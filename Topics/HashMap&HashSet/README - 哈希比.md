## **4. 哈希表（HashMap & HashSet）**
**📌 內容**：利用哈希函數進行快速查找  
**🔹 常見題型**：
- **哈希表計數**：頻率統計、唯一元素
- **哈希映射**：字母異位詞、鍵值對應
- **雙哈希**：查找重複數據
